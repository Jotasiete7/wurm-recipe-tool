# A Guilda - Wurm Recipes Guide 🍳

> Parte do ecossistema **A Guilda**.

Uma ferramenta interativa e responsiva para **Wurm Online** que permite aos jogadores navegar, buscar e filtrar o vasto catálogo de receitas culinárias do jogo.

![Wurm Online](https://img.shields.io/badge/Game-Wurm%20Online-gold)
![Tech](https://img.shields.io/badge/Built%20With-React%20%7C%20Vite%20%7C%20Tailwind-blue)

## 📋 Funcionalidades

- **Busca Inteligente:** Encontre receitas pelo nome ou pelos ingredientes necessários.
- **Filtros Avançados:** Refine sua busca por:
  - Habilidade (ex: Hot Food Cooking, Baking)
  - Recipiente (ex: Pottery Bowl, Saucepan)
  - Utensílio de Cozinha (ex: Oven, Campfire)
- **Detalhes da Receita:** Visualize ingredientes obrigatórios e requisitos de ferramentas.
- **Estatísticas:** Gráfico visual da distribuição de receitas por habilidade.
- **Multi-idioma:** Interface disponível em Inglês, Português e Russo.
- **Design:** Tema escuro "Dark Mode" inspirado na estética do jogo.

## 🚀 Como Usar

1. **Acesse a ferramenta:** (Link da sua page aqui, ex: `https://wurm-recipes.pages.dev`)
2. **Busque:** Digite o nome de um ingrediente que você possui (ex: "meat", "corn") na barra de busca.
3. **Filtre:** Utilize os menus para selecionar qual ferramenta você quer usar (ex: "Frying Pan").
4. **Detalhes:** Clique no cartão da receita para ver a lista completa de ingredientes e instruções.

## 🌍 Tradução e Termos

Para manter a consistência com o jogo e facilitar a busca na Wiki oficial:
- **Nomes de Itens e Receitas:** Mantidos em **Inglês** (original do jogo).
- **Interface (Botões, Menus, Títulos):** Traduzida para Português, Inglês e Russo.

## 🛠️ Instalação e Desenvolvimento

Se você quiser rodar este projeto em seu computador:

1. Clone o repositório:
   ```bash
   git clone https://github.com/seu-usuario/a-guilda-recipes.git
   ```

2. Instale as dependências:
   ```bash
   npm install
   ```

3. Rode o servidor de desenvolvimento:
   ```bash
   npm run dev
   ```
